# P02: Repositori de GitHub

Ara ja sabem que GitHub ens proporciona una plataforma molt potent per gestionar la documentació dels nostres projectes.  
Per això, en aquest tercer projecte, al repositori que haureu creat a la **tasca 0**, caldrà que:

- Tingueu **una carpeta per a cadascuna de les activitats amb lliurament**.  
- A dins de cada carpeta hi hagi les **guies, documentacions, configuracions, etc.** que es demanin.

Per una correcta gestió del repositori, és molt important mantenir un **criteri uniforme**:

---

## ✔️ Estructura recomanada del repositori

### **1. Arxiu principal `README.md`**
Ha d’incloure:
- Presentació del projecte  
- Enllaços a totes les carpetes i activitats  

---

### **2. Carpetes per a les tasques**
Exemple:  
- `tasca01/`  
- `tasca02/`  
- `tasca03/`  

A cada carpeta:
- Un arxiu `README.md` amb:
  - L’enunciat o descripció  
  - L’enllaç a l’arxiu o arxius que continguin la solució  

---

### **3. Carpetes per als productes**
Per als productes finals amb lliurament:
- Crear una carpeta amb el nom:  
  - `producte04/`, `producte05/`, etc.
- Seguir **el mateix criteri** que amb les tasques:
  - README.md d’explicació  
  - Enllaços als fitxers del lliurament
