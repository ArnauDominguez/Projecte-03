# T09: ODS i el sector IT: el costat sostenible d’EverPia (grups)

EverPia ha rebut una petició del client **DigiCore**, preocupat per l’impacte ambiental i social dels serveis tecnològics que contracta (consum energètic dels servidors, obsolescència del maquinari, teletreball, igualtat de gènere, etc.).

L’empresa demana a EverPia un **breu informe** per comprendre com els **Objectius de Desenvolupament Sostenible (ODS)** s’apliquen al sector IT i a les seves operacions diàries.
